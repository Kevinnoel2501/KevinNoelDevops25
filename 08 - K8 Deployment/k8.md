# Kubernetes Deployment

- Created a **namespace**: `random-app`.  
- Deployed a **ClusterIP service** and **Deployment** with 3 replicas.  
- Set **readiness and liveness probes**.  
- Configured resource usage per container: requests — CPU 100m, memory 128Mi; limits — CPU 500m, memory 256Mi.  
- Deployment pulls images from **ACR**.  
- Faced **ImagePullbackoff pod failures** due to wrong image names.  
- Debugged with `kubectl describe` and `kubectl logs`.  
- Redeployed successfully using `kubectl rollout restart deployment`.
